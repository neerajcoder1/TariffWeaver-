# 🕸️ TariffWeaver: AI for Enterprise Cost Intelligence & Autonomous Action

**Winning Prototype for Autonomous Procurement & Spend Intelligence.**

TariffWeaver is a professional AI system designed to go beyond dashboards. It continuously monitors global enterprise data and trade signals (NewsAPI), identifies **"cost leakage"** patterns caused by changing tariffs, and initiates **autonomous corrective actions** through quantifiable financial playbooks.

---

## ⚡ Problem Statement
In a world of rapidly shifting trade policies, enterprises face massive **cost leakage** due to reactive procurement cycles. TariffWeaver autonomously monitors operational data and market signals to protect margins before the financial hit lands.

## 🤖 The Spend Intelligence Logic
Our architecture uses specialized agents designed for **Autonomous Action**:

1.  **⚖️ Regulatory (Spend Intelligence Agent)**: Digs into procurement and vendor data (BOM entries) to find rate optimization opportunities after new trade signals emerge.
2.  **🛠️ Engineering (Resource Optimization Agent)**: Generates actionable **"redesign playbooks"** and suggests sourcing shifts to avoid approaching cost breaches.
3.  **💰 FinOps (Cost Intelligence Agent)**: Provides **variance analysis with root-cause attribution** (the tariff event), calculating exactly why and how the enterprise will save thousands of dollars anually.

## 🚀 Key Features for Judges
- **Beyond Dashboards**: Not just a chart—an autonomous agent pulse that coordinates the full decision cycle.
- **Autonomous Triggering**: Uses live market signals as the primary trigger for procurement workflows.
- **Quantifiable Outcome**: Direct financial impact modeling ($200k+ annual savings identified per component).
- **Executive Ready**: The UI features an **"Executive Approval → Procurement Workflow Trigger"** status, showing it's ready for immediate downstream integration.

## 🚀 Key Features
- **Live Cloud News**: Uses Google-level NewsAPI integration to scan the world's trade headlines. 
- **Predictive Mapping**: Maps high-level trade events (like EV trade wars) to micro-component impacts (like Magnets).
- **Glassmorphic UI**: High-impact design with live agent logs and a "Confidence Score" for decision-ready intelligence.
- **Fail-Safe Demo Mode**: Integrated "Mock Demo" bypasses API rate limits for a zero-fail presentation.

## 🛠️ How to Launch
1.  Open `index.html` in your browser.
2.  (Optional) Paste your **NewsAPI Key** (from `newsapi.org`) and click **Scan Global News**.
3.  If no API key is available, click **Load Mock Demo** for a pre-configured scenario.
4.  Watch the **Agent Pulse** logs as the agents coordinate their redesign.

## 📦 Tech Stack
- **Structure**: Semantic HTML5
- **Style**: Custom CSS3 with Backdrop filters (Glassmorphism)
- **Logic**: Vanilla JavaScript (The choice for zero-latency demos!)
- **Data**: NewsAPI REST Integration

## 🛠️ Setup & Installation
To run this project locally for testing or judge review:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/neerajcoder1/TariffWeaver.git
    cd TariffWeaver
    ```
2.  **Open the Application**:
    Simply open the `index.html` file in any modern web browser (Chrome, Edge, or Firefox).
    *Tip: You can use the "Live Server" extension in VS Code for the best experience.*
3.  **Active the Demo**:
    *   **Option A (Live API)**: Paste your NewsAPI key from [newsapi.org](https://newsapi.org/) into the input field and click **Scan Global News**.
    *   **Option B (Offline Demo)**: Click **Load Mock Demo** to run the pre-configured high-impact trade scenario.
4.  **Run Analysis**: Click **Run Multi-Agent Analysis** to see the agents in action.
